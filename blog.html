Important Questions:-

What is the DOM?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a document as a tree of nodes, where each node corresponds to a part of the document (such as an element, attribute, or text). JavaScript can manipulate the DOM to dynamically change the content, structure, and style of a webpage.

How is the DOM structured?

The DOM is structured like a tree, starting with the root node (the document object), followed by branches that represent elements like html, head, body, and all child elements within these tags. Each element in the DOM tree is a node, and the relationships between elements (parent, child, sibling) form the tree structure.

How do you select DOM elements using JavaScript?

You can select DOM elements using several methods:
document.getElementById('id'): Selects an element by its id.
document.getElementsByClassName('className'): Selects all elements with a particular class.
document.getElementsByTagName('tag'): Selects elements by their tag name.
document.querySelector('selector'): Selects the first element that matches a CSS selector.
document.querySelectorAll('selector'): Selects all elements that match a CSS selector.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you primarily interact with the element's properties and methods in JavaScript. Attributes can be modified using the setAttribute() method, which allows you to change existing attributes or add new ones, such as modifying the src of an image or the href of a link. Additionally, you can access certain attributes directly through properties associated with the element. For styles, you can use the style property to change CSS styles directly. This allows you to modify aspects like colors, sizes, and positioning by setting individual CSS properties. By dynamically altering attributes and styles, you can create interactive and responsive web pages, enhancing user experience and visual appeal.
